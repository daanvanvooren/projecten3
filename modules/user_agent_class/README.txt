INTRODUCTION
------------
This module help you to set css class in body depends from header user agent,
device.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/user_agent_class

 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/search/user_agent_class

REQUIREMENTS
------------
No special requirements.

INSTALLATION
------------

 * Copy user_agent_class folder to modules directory (usually /modules/).

 * At admin/modules enable the User Agent Module.

CONFIGURATION
-------------

 * User agent (Browser)
   http://example.com/admin/config/system/user-agent

 * Device (Phone, Table, etc)
   http://example.com/admin/config/system/device

 * Current User-Agent
   http://example.com//admin/config/system/check-user-agent

 * Responsibility (Frontend/Backend)
   http://example.com/admin/config/system/provide

   An example Header -> User Agent:
   - Chrome
   Mozilla/5.0 (Windows NT 6.1; Win64; x64) AppleWebKit/537.36 (KHTML, like
   Gecko) Chrome/60.0.3112.90 Safari/537.36
